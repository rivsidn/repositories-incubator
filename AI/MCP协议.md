
`MCP` 提供AI模型和不同数据源和工具的标准链接方式.

| 角色        | 功能                                       |
|-------------|--------------------------------------------|
| MCP Host    | 能够通过MCP协议访问特定数据的工具          |
| MCP Clients | 客户端，与服务器维持1:1关系                |
| MCP Servers | 服务器，通过MCP 服务器暴露特定能力给客户端 |
